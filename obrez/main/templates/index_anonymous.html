{{ SITE.name }} — отличный способ создания красивых и удобных ссылок

{{ SITE.name }} — новый сервис для сокращения ссылок, создан для российских пользователей блогосферы и социальных сетей.

Сервис предоставляет пользователям возможность сократить длинный URL в несколько раз и повсеместно без каких–либо ограничений использовать его, просматривать статистику переходов и управлять активностью сгенерированных ссылок.

В дополнение к самому сервису сокращения URL, {{ SITE.name }} генерирует уникальный QR Code (двухмерный штрихкод) для каждой ссылки, который идентифицирует полученный результат на внешних устройствах, например мобильный телефон.

Qlink ссылка — это удобно и эргономично

Короткие ссылки намного удобнее в использовании, они выглядят намного лучше нежели чем беспорядочно длинный URL, легко читаются и запоминаются, меньше «весят» (от 15 до 21 байта), максимальная длина сокращённой ссылки не превышает 22 символа.

Пример:

На сайте http://ru.wikipedia.org cсылки после копирования выглядят, так:

http://ru.wikipedia.org/wiki/%D0%93%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%D1%8B_%D0%B8_%D1%81%D0%BA%D0%B0%D0%BD%D0%B5%D1%80%D1%8B_%D1%88%D1%82%D1%80%D0%B8%D1%85%D0%BA%D0%BE%D0%B4%D0%BE%D0%B2

Однако та же самая ссылка, после преобразованния в Qlink, выглядит, вот так: http://{{ SITE.domain }}/S89

Следите за продолжением...
